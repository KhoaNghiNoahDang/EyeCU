# EyeCU
EyeCU là hệ điều hành nhận thức không gian (Ambient Intelligence), tích hợp API VNPT để tối ưu trải nghiệm "không chạm". Hệ thống tự động điều phối hạ tầng, giám sát nội trú và thông dịch chỉ số y tế thời gian thực. EyeCU chuyển dịch mô hình y tế từ "bị động hậu kiểm" sang "chủ động dự báo", giúp giải phóng sức tải cho hệ thống y tế hiện đại.
